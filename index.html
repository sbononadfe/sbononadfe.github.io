Mis Proyectos

Sitios Web con Apache 2

La tarea consistió en configurar Apache 2 en una máquina virtual para alojar dos sitios web independientes de forma simultánea, utilizando la técnica de Virtual Hosts

Programación Asistida por Copilot/Gemini

Práctica de utilizar una inteligencia artificial integrada en el editor de código para generar, completar, depurar y explicar código en tiempo real, acelerando el flujo de trabajo del desarrollador.

Copias de seguridad

AOMEI Backupper es un software de copia de seguridad y recuperación que permite a los usuarios crear fácilmente imágenes del sistema, clonar discos y respaldar archivos o particiones para protegerse contra la pérdida de datos.

Configuración carpeta compartida con Samba y cuotas de disco para grupos

La tarea consistió en configurar una carpeta de red accesible mediante Samba, implementando a su vez cuotas de disco para restringir el espacio total de almacenamiento permitido para un grupo de usuarios específico.

Instalación De Servicio DNS En Ubuntu Server

La tarea consistió en instalar y configurar el servicio BIND9 en un Ubuntu Server para implementar un servidor DNS capaz de resolver nombres de dominio tanto para una zona local como de forma inversa.

Configuración de Servidores DNS y Web en Cisco Packet Tracer

Esta práctica consiste en configurar una red LAN en Packet Tracer con servicios DHCP, HTTP y DNS para que los clientes puedan acceder a una página web alojada localmente usando un nombre de dominio.
